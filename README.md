nwtg-kata
=========

Rails app for an atdd kata

Dependencies - 

Ruby1.9.2 or greater
'bundler' gem

Windows:

DevKit installed for Ruby

OSX:

XCode or standalone GCC Compiler installed

Ubuntu/other Debian Linux:
Packages:
libxml2-dev
libxslt-dev
libsqlite3-dev
libv8-3.7.12.22

Running:

from the app root: 

bundle exec rails s

if you are asked about missing gems, run:

bundle install
